HTML As páginas HTML são documentos de texto O HTML usa tags, que são caracteres que ficam dentro de colchetes angulares.
Eles agem como recipientes e dizem algo sobre as informaçãoes que fica entre eles 		As tags geralmente são chamadas de elementos. As tags geralmente vêm em pares.
A tag de abertura denota o inicio de um conteúdo; a etiqueta de fechamento denota o fim.		 As tags de abertura podem conter atributos, que nos informam mais sobre o conteúdo desse elemento. Os atributos requerem um nome e um valor. Para aprender HTML você precisa saber o que são tags disponiveis para você usar, o que eles fazem e onde eles podem ir .
